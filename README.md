# Coding session novembre

## Installation

Dépendances :
 * [Node.js] v≥6.

```sh
yarn install
```

## Lancer le site

L'application est une application [React] dans le dossier `src`.

```
yarn start
```

## Lancer les tests

```
yarn test
```

[Node.js]: https://nodejs.org/en/
[React]: https://facebook.github.io/react/
